# Firefox Add-ons Store Description (ready to paste)

## Title
Cardmarket Purchases → CSV / ManaBox / Moxfield / Archidekt

## Summary
Export your Cardmarket purchase history to CSV or import formats for ManaBox, Moxfield, and Archidekt — with Scryfall set codes & IDs.

## Description
This extension helps you export your Cardmarket purchases (including detailed order data and per-card line items) into formats you can use in Google Sheets/Excel or import directly into popular collection/deck apps.

### Key features
• Export current page or all purchase pages  
• Detailed CSV export for spreadsheets and analysis  
• ManaBox import CSV (includes Set Code + Scryfall ID)  
• Moxfield import  
• Archidekt import  
• Optional shipping cost splitting for ManaBox (none / equal per card / proportional)  
• Progress display with optional log while exporting  

### Notes
• You must be logged into Cardmarket.  
• This project is not affiliated with Cardmarket, Scryfall, ManaBox, Moxfield, or Archidekt.  

## Permissions rationale
• Access to Cardmarket pages: needed to read your order pages and export data  
• Downloads: needed to save the generated export file to your computer  

## Privacy
This extension runs locally in your browser. It does not sell or share your data.  
Scryfall lookups are performed to enrich cards with set codes and IDs.
